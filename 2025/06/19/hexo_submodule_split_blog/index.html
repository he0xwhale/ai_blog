本文记录了一个完整的实践过程，包含遇到的问题、原因分析和最终的解决方法。

  • 遇到的问题包括：子模块复用导致冲突、分支不同步、远程混乱、推送失败等。

六、其他注意事项

  • .git/modules/ 是 Git 存放子模块缓存数据的地方，必须清除干净；
  • 使用多个项目共用子模块时，应考虑使用不同分支，或完全复制主题；
